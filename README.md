
# Performance Quant - 量化績效分析系統

**Performance Quant** 是一款專為台股交易者設計的自動化績效追蹤工具。本系統透過「Vibe Coding」理念開發，結合 AI 輔助邏輯與前端視覺化技術，幫助投資者精確掌握每一筆交易的稅後淨利與長期資產走勢。

## 🚀 核心功能

*   **台股成本自動化計算**：內建台股手續費（預設 28 折）與證券交易稅（0.3%）計算邏輯，精準還原真實損益。
*   **多維度績效指標 (KPI)**：即時顯示本日、本月、今年及累積總損益，並包含勝率、獲利因子 (PF)、最大回撤 (MDD) 與期望值分析。
*   **動態視覺化圖表**：
    *   **損益曲線**：追蹤資產累計餘額的波動狀況。
    *   **策略勝率圓餅圖**：分析不同交易標籤（如：動能突破、價值投資）的勝率分佈。
*   **雲端同步與驗證**：整合 Firebase Firestore 實時資料庫，支援 Email 與 Google 帳號快速登入，數據隨時隨地同步。
*   **智慧標的搜尋**：內建台股上市、上櫃及 ETF 股名對照表，輸入代碼即可自動對應名稱。
*   **完整交易管理**：支援新增、編輯、刪除紀錄，並具備自定義每頁筆數的分頁機制。

## 🛠 關鍵技術棧

*   **前端框架**：React (Hooks, useMemo, useEffect)
*   **資料庫與驗證**：Firebase (Firestore & Auth)
*   **圖表插件**：Recharts
*   **圖示庫**：Lucide-React
*   **樣式處理**：Tailwind CSS 搭配玻璃擬態 (Glassmorphism) UI

## 📝 快速開始

1.  **安裝依賴項目**：
    ```bash
    npm install
    ```
2.  **環境變數設定**：
    請在專案根目錄建立 `.env` 檔案並加入你的 Firebase 金鑰。
3.  **啟動專案**：
    ```bash
    npm start
    ```

## 📊 損益計算邏輯

系統採用以下公式確保計算精確度：
*   **買入成本**：`價格 * 股數 * 0.001425 * 折扣` (最低 1 元)
*   **賣出成本**：`價格 * 股數 * 0.001425 * 折扣 + 價格 * 股數 * 0.003`
*   **淨損益**：`(賣出價 - 買入價) * 股數 - 總成本`
